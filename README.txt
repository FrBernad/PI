A continuacion se daran las instrucciones de como generar el ejecutable y como ejecutar el programa. Previamente debera asegurarse de haber descomprimido el archivo BDDT.zip, y de tener todo su contenido dentro del directorio actual (archivos con extension .c y .h).

PARA GENERAR EJECUTABLE:

1) Ejecutar el comando "make" en consola, lo que dara origen a un nuevo archivo ejectuable llamado BDDT.


PARA EJECUTAR EL PROGRAMA:

1) A continuacion, se debera introducir el siguiente comando en consola "./BDDT path/to/nacimientos.csv path/to/provincias.csv" (Cerciorarse que los archivos se 
encuentren en el orden indicado) . Esto generara 3 archivos con extension ".csv" que contendran la informacion respectiva de cada uno de los querys solicitados. 


ADICIONAL:

-Para eliminar los archivos generados (a excepcion de los Querys), ejecutar el comando "make clean".
